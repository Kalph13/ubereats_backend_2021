# Uber Eats

- id
- createdAt
- updatedAt
- email
- password
- role (client|owner|delivery)

## User CRUD:

- Create Account
- Log In
- See Profile
- Edit Profile
- Verify Email

- coverImage

- Create Dish
- Edit Dish
